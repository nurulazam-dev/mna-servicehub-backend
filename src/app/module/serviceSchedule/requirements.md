1. createServiceSchedule => a serviceProvider can create 3 schedules in a day. every schedule duration will be 3 hours. there have 15 minutes gap after every schedule. the schedule date will be future date, not accepted previous date.

2. getMySchedules => a serviceProvider can see her all schedules.

3. getScheduleByDate => a serviceProvider can see his own schedules by selected a date. admin and manager can see anyone schedules by selected a date.

4. getServiceSchedules => An admin or manager can see the all service schedules.

5. scheduleById => a serviceProvider(own), admin and manager can see the schedule by id.
